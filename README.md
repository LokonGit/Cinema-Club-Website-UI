## Installation & Setup

### Prerequisites
- Node.js (v16 or higher)
- npm 

### Steps

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run development server**
   ```bash
   npm run dev
   ```

3. **Open in browser**
   Navigate to `http://localhost:5173`
